Features
========

WiFi-Lab Controller provides a controlled environment for demonstrating:

Network Control Features
------------------------

- Safe Wi-Fi reset (down/up wlan0)
- Automatic monitor mode setup (airmon-ng)
- Channel switching
- NAT enablement

Wireless Simulation Tools
-------------------------

- Duplicate/Fake AP using hostapd
- DNS redirection using dnsmasq
- Basic phishing redirection lab (educational only)

Wireless Scanning
-----------------

- Detect 2.4 GHz and 5 GHz networks
- Parse airodump CSV output
- Automatic stop timer
- View ESSID, BSSID, channel, and band

Deauthentication Demonstration
------------------------------

Runs an authorized deauthentication demonstration using aireplay-ng.

Educational Use Cases
---------------------

- Cybersecurity lab exercises  
- CTF training  
- Demonstrating wireless vulnerabilities  
- Learning network modes and scanning
